🎯 Что такое навык?

Навык (skill) - это расширение для OpenClawGotchi, которое добавляет новую функциональность. Навыки могут:

  • 🌤️ Показывать погоду
  • 📧 Проверять email
  • 🎵 Управлять музыкой
  • 📈 Отслеживать задачи
  • 🤣 Отправлять шутки
  • 💹 Показывать курсы валют
  • 📝 Управлять заметками
  • И многое другое!

Каждый навык - это отдельный Python-модуль, который можно легко включить или отключить.

📁 Структура навыка

skills/
├── __init__.py
├── base.py          # Базовый класс навыка
├── weather/         # Пример навыка: погода
│   ├── __init__.py
│   ├── skill.py     # Основной код навыка
│   └── config.json  # Конфигурация
└── your_skill/      # Ваш новый навык
    ├── __init__.py
    ├── skill.py
    └── config.json

🔨 Создание простого навыка

Шаг 1: Создайте папку навыка

mkdir skills/hello
cd skills/hello

Шаг 2: Создайте config.json

{
  "name": "hello",
  "version": "1.0.0",
  "description": "Приветствует пользователя",
  "author": "Ваше Имя",
  "enabled": true,
  "commands": ["hello", "hi"],
  "permissions": []
}

Шаг 3: Создайте __init__.py

from .skill import HelloSkill

__all__ = ['HelloSkill']

Шаг 4: Создайте skill.py

from skills.base import Skill

class HelloSkill(Skill):
    """Навык для приветствия"""
    
    def __init__(self, bot):
        super().__init__(bot)
        self.name = "hello"
        self.description = "Приветствует пользователя"
    
    async def handle_hello(self, message):
        """Обработчик команды /hello"""
        user_name = message.from_user.first_name
        greeting = f"Привет, {user_name}! 👋"
        
        # Отправляем ответ
        await message.answer(greeting)
        
        # Показываем на дисплее
        self.bot.display.show("happy")
        
        # Логируем использование
        self.log(f"Приветствовал {user_name}")
    
    def get_commands(self):
        """Возвращает список команд навыка"""
        return {
            'hello': self.handle_hello,
            'hi': self.handle_hello
        }

Шаг 5: Активируйте навык

# В главном боте или через команду
/enable hello

Шаг 6: Используйте навык

/hello
# Бот ответит: Привет, Иван! 👋

🎨 Расширенные возможности

Работа с API

import requests

class WeatherSkill(Skill):
    async def get_weather(self, city):
        url = f"https://api.openweathermap.org/data/2.5/weather?q={city}&appid={API_KEY}"
        response = requests.get(url)
        data = response.json()
        
        temp = data['main']['temp']
        return f"Погода в {city}: {temp}°C"

Использование памяти

class MemorySkill(Skill):
    async def remember_fact(self, fact):
        # Сохраняем в долгосрочную память
        await self.bot.memory.remember(
            category="facts",
            content=fact,
            importance=0.8
        )

Планировщик задач

class ReminderSkill(Skill):
    async def set_reminder(self, text, delay_minutes):
        # Ставим задачу через N минут
        await self.bot.scheduler.schedule(
            name=f"reminder-{time.time()}",
            run_in_minutes=delay_minutes,
            message=f"Напоминание: {text}"
        )

Дисплей

# Показать лицо
self.bot.display.show_face("happy")

# Показать текст
self.bot.display.show_text("Привет!")

# Показать статистику
self.bot.display.show_stats()
