📋 Требования

Оборудование

  • Raspberry Pi Zero 2W или выше (512MB RAM минимум)
  • Waveshare 2.13" E-Ink дисплей V4 или совместимый
  • Карта microSD минимум 16GB Class 10
  • Питание 5V 2A (для Zero 2W) или 5V 3A (для Pi 3/4)

Программное обеспечение

  • Raspberry Pi OS Lite (64-bit) - рекомендуется
  • Python 3.9+
  • Git
  • pip (Python package manager)

🚀 Быстрая установка

Шаг 1: Подготовка Raspberry Pi

Установите Raspberry Pi OS Lite на карту microSD. Загрузите систему и подключитесь через SSH.

Шаг 2: Обновление системы

sudo apt update && sudo apt upgrade -y

Шаг 3: Установка зависимостей

sudo apt install -y python3-pip python3-venv git

# Включите SPI и I2C для E-Ink дисплея
sudo raspi-config

В raspi-config выберите: Interface Options → SPI → Enable и Interface Options → I2C → Enable

Шаг 4: Клонирование репозитория

git clone https://github.com/openclawgotchi/openclawgotchi.git
cd openclawgotchi

Шаг 5: Создание виртуального окружения

python3 -m venv venv
source venv/bin/activate

Шаг 6: Установка Python-зависимостей

pip install -r requirements.txt

Шаг 7: Настройка окружения

cp .env.example .env
nano .env

Шаг 8: Получение Telegram Bot Token

  1. Откройте Telegram и найдите @BotFather
  2. Отправьте команду /newbot
  3. Следуйте инструкциям для создания бота
  4. Скопируйте полученный токен
  5. Вставьте токен в .env файл:
TELEGRAM_TOKEN=ваш_токен_здесь

Шаг 9: Запуск бота

python src/main.py

✅ Готово! Ваш бот теперь работает!

🔌 Подключение E-Ink дисплея

Waveshare 2.13" E-Ink V4 Pinout

Display    →  Raspberry Pi
VCC        →  3.3V
GND        →  GND
DIN/CLK    →  GPIO 11 (SCLK)
DOUT/MOSI  →  GPIO 10 (MOSI)
CS         →  GPIO 8  (CE0)
DC         →  GPIO 25
RES/RST    →  GPIO 17
BUSY       →  GPIO 24

Проверка подключения

После подключения запустите тест дисплея:

python tools/test_display.py

⚙️ Конфигурация

.env файл

Основной файл конфигурации - .env. Вот основные параметры:

# Telegram Bot
TELEGRAM_TOKEN=ваш_токен_здесь

# Discord Bot (опционально)
# DISCORD_TOKEN=ваш_токен_здесь

# OpenAI API (для AI-функций)
OPENAI_API_KEY=ваш_ключ_здесь

# Имя бота
BOT_NAME=OpenClawGotchi

# Владелец (ваш Telegram username)
OWNER_USERNAME=ваш_username

# Логирование
LOG_LEVEL=INFO

Настройка автозапуска

Для автозапуска бота при загрузке системы используйте systemd:

sudo cp setup/gotchi-bot.service /etc/systemd/system/
sudo systemctl enable gotchi-bot
sudo systemctl start gotchi-bot

Проверка статуса

sudo systemctl status gotchi-bot

🐛 Устранение проблем

Бот не запускается

  • Проверьте что токен в .env правильный
  • Убедитесь что все зависимости установлены: pip install -r requirements.txt
  • Проверьте логи: tail -f logs/gotchi.log

Дисплей не работает

  • Проверьте подключение проводов
  • Убедитесь что SPI включён: raspi-config
  • Протестируйте дисплей: python tools/test_display.py

Бот отвечает медленно

  • Проверьте нагрузку на CPU: htop
  • Убедитесь что достаточно свободной RAM
  • Попробуйте отключить тяжёлые навыки
