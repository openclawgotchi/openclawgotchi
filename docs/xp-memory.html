📈 Система опыта (XP)

OpenClawGotchi имеет систему развития, похожую на игры. Бот получает опыт (XP) за различные действия и повышает свой уровень.

Как получить XP?

  • 💬 Каждое сообщение от пользователя: +10 XP
  • 🎯 Выполнение команды: +25 XP
  • ⚡ Использование навыка: +50 XP
  • 🧠 Запоминание факта: +75 XP
  • 🔧 Саморазвитие (изменение кода): +200 XP
  • 📅 Каждый день активности: +100 XP

Уровни

Уровень 1:  0-1000 XP    (Новичок)
Уровень 2:  1000-2500 XP  (Ученик)
Уровень 3:  2500-5000 XP  (Подмастерье)
Уровень 4:  5000-10000 XP (Эксперт)
Уровень 5:  10000-25000 XP (Мастер)
Уровень 6:  25000-50000 XP (Гроссмейстер)
Уровень 7:  50000-100000 XP (Легенда)
Уровень 8:  100000+ XP     (Мифический)

Бонусы за уровни

С каждым уровнем бот получает новые возможности:

  • Уровень 2: Доступ к расширенной памяти
  • Уровень 3: Возможность редактировать код
  • Уровень 4: Доступ к системным функциям
  • Уровень 5: Возможность обучать других ботов
  • Уровень 6+: Полный доступ ко всем функциям

💾 Долговременная память

OpenClawGotchi имеет мощную систему памяти, которая хранит важную информацию indefinitely.

Типы памяти

🧠 Краткосрочная память (Working Memory)

Хранит контекст текущего разговора. Автоматически очищается через 24 часа.

Пример: "Мы говорили о Python yesterday"
→ Хранится 24 часа

📚 Долгосрочная память (Long-term Memory)

Хранит важную информацию indefinitely. Использует SQLite базу данных.

Пример: "Мой любимый цвет - синий"
→ Хранится навсегда
→ Можно извлечь позже

⚡ Эпизодическая память (Episodic Memory)

Запоминает важные события и достижения.

Пример: "15 марта 2024: Бот достиг уровня 5"
→ Записывается в лог достижений

🔧 Семантическая память (Semantic Memory)

Хранит знания и факты об мире.

Пример: "Python - язык программирования"
→ Хранится как знание

📝 Команды памяти

/remember

Запомнить информацию на долгий срок.

/remember У меня есть собака по имени Шарик
✅ Запомнено!

/recall

Извлечь информацию из памяти.

/recall собака
🧠 У тебя есть собака по имени Шарик

/forget

Забыть конкретную информацию.

/forget Шарик
🗑️ Забыто!

/memory

Показать статистику памяти.

📊 Память:
- Фактов: 47
- Событий: 12
- Знаний: 156
- Использовано: 2.4 MB

/search

Поиск в памяти по ключевому слову.

/search Python
🔍 Результаты:
- Python изучается 3 месяца
- Python использовался в проекте X
- Python любимый язык программирования

🎯 Как память используется?

💬

В диалогах

Бот помнит контекст разговора и может ссылаться на предыдущие сообщения.

📅

В напоминаниях

Помнит ваши задачи и важные даты.

🎭

В настроении

Запоминает ваши предпочтения и адаптируется под них.

📈

В развитии

Использует прошлый опыт для обучения.

🔧 Для разработчиков

Использование памяти в навыках

from skills.base import Skill

class MySkill(Skill):
    async def process_message(self, message):
        # Сохранить в память
        await self.bot.memory.remember(
            category="preferences",
            content="Пользователь любит Python",
            importance=0.8
        )
        
        # Извлечь из памяти
        facts = await self.bot.memory.recall("Python")
        
        # Поиск по памяти
        results = await self.bot.memory.search("программирование")

Структура данных памяти

  "id": "uuid-везде",

  "content": "Текст для запоминания",
  "importance": 0.8,
  "created_at": "2024-03-09T12:00:00",
  "access_count": 5,
  "last_accessed": "2024-03-09T14:30:00"
}

💡 Советы

  • ✅ Запоминайте важное - используйте /remember для важных фактов
  • ✅ Проверяйте память - регулярно используйте /memory для просмотра статистики
  • ✅ Очищайте лишнее - используйте /forget для удаления устаревшей информации
  • ✅ Активно общайтесь - больше общения = больше XP = быстрее развитие
  • ✅ Используйте навыки - это даёт больше XP чем простые сообщения
