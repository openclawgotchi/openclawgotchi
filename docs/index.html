🚀 Быстрый старт

1. Клонируйте репозиторий

git clone https://github.com/openclawgotchi/openclawgotchi.git
cd openclawgotchi

2. Установите зависимости

pip install -r requirements.txt

3. Настройте окружение

cp .env.example .env
nano .env

4. Запустите бота

python src/main.py

Готово! Ваш бот теперь работает и готов к общению!

🖥️ Оборудование

Для работы OpenClawGotchi потребуется:

  • Raspberry Pi Zero 2W (512MB RAM минимум)
  • Waveshare 2.13" E-Ink дисплей V4 или совместимый
  • Raspberry Pi OS Lite (64-bit)
  • Карта microSD минимум 16GB Class 10
